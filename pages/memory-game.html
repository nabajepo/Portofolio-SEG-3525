Design 2 : Jeu de mémoire

Cette étude de cas présentera un futur jeu de mémoire interactif.

Le projet consistera à développer un jeu de mémoire permettant aux utilisateurs d’améliorer leur concentration et leurs capacités de mémorisation grâce à une interface simple, amusante et intuitive. Le design mettra l’accent sur une bonne expérience utilisateur, une navigation claire et un retour visuel efficace.

Cette page est un espace réservé. Le design complet sera ajouté plus tard dans le semestre.
